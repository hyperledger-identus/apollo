# Apollo - AES

[![Kotlin](https://img.shields.io/badge/kotlin-1.7.21-blue.svg?logo=kotlin)](http://kotlinlang.org)

Apollo RSA is Kotlin Multiplatform library for RSA & RSA/PSS signature.

## Supported Key Size

- 1024
- 2048
- 3072
- 4096

## Supported Signature

- RSA
- RSA/PSS
