# Apollo - Hashing
[![Kotlin](https://img.shields.io/badge/kotlin-1.6.21-blue.svg?logo=kotlin)](http://kotlinlang.org)

Apollo Hashing is Kotlin Multiplatform Hashing library containing:
- MD2
- MD4
- MD5
- SHA0
- SHA1
- SHA-2 family
- SHA-3 family
- Blake
- Blake2b
- Blake2s

## Supported Targets
- WindowsX86
- WindowsX64
- iOS
- tvOS
- watchOS
- macOS
- Android
- JVM
- JS
