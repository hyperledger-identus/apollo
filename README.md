# Apollo
[![Kotlin](https://img.shields.io/badge/kotlin-1.6.21-blue.svg?logo=kotlin)](http://kotlinlang.org)

A cryptography lib built with Kotlin Multiplatform with support for the following targets:
- JS
- iOS
- Android
- JVM
